<h1 class="firstHeading" id="firstHeading" lang="en">Kilo-</h1> <div class="mw-body-content" id="bodyContent">
<div class="noprint" id="siteSub">From Wikipedia, the free encyclopedia</div> <div id="contentSub"></div>

<p><b>Kilo</b> is a <a class="mw-redirect" href="/wiki/Decimal_prefix" title="Decimal prefix">decimal</a> <a href="/wiki/Metric_prefix" title="Metric prefix">unit prefix</a> in the <a href="/wiki/Metric_system" title="Metric system">metric system</a> denoting multiplication by one <a class="mw-redirect" href="/wiki/Thousand" title="Thousand">thousand</a> (10<sup>3</sup>). It is used in the <a href="/wiki/International_System_of_Units" title="International System of Units">International System of Units</a> where it has the unit symbol <b>k</b>, in lower case.
</p><p>The prefix <i>kilo</i> is derived from the <a href="/wiki/Greek_language" title="Greek language">Greek</a> word <span lang="el" title="Greek language text">χίλιοι</span> (<i lang="el-Latn" title="Greek language text">chilioi</i>), meaning "thousand". It was originally adopted by <a href="/wiki/Antoine_Lavoisier" title="Antoine Lavoisier">Antoine Lavoisier</a>'s research group in 1795, and introduced into the metric system in France with its establishment in 1799.
</p><p>In 19th century English it was sometimes spelled chilio, in line with a <a href="/wiki/Linguistic_purism" title="Linguistic purism">puristic</a> opinion by <a href="/wiki/Thomas_Young_(scientist)" title="Thomas Young (scientist)">Thomas Young</a><sup class="reference" id="cite_ref-Brewster_1832_1-0"><a href="#cite_note-Brewster_1832-1">[1]</a></sup><sup class="reference" id="cite_ref-Dingler_1823_2-0"><a href="#cite_note-Dingler_1823-2">[2]</a></sup>
</p>

<h2><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kilo-&amp;action=edit&amp;section=1" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>one <a href="/wiki/Kilogram" title="Kilogram">kilogram</a> (kg) is 1000 <a href="/wiki/Gram" title="Gram">grams</a></li>
<li>one <a href="/wiki/Kilometre" title="Kilometre">kilometre</a> (km) is 1000 <a href="/wiki/Metre" title="Metre">metres</a></li>
<li>one <a class="mw-redirect" href="/wiki/Kilojoule" title="Kilojoule">kilojoule</a> (kJ) is 1000 <a href="/wiki/Joule" title="Joule">joules</a></li>
<li>one kilolitre (kL) is 1000 <a href="/wiki/Litre" title="Litre">litres</a></li>
<li>one kilobaud (kBd) is 1000 <a href="/wiki/Baud" title="Baud">baud</a></li>
<li>one kilohertz (kHz) is 1000 <a href="/wiki/Hertz" title="Hertz">hertz</a></li>
<li>one <a href="/wiki/Kilobit" title="Kilobit">kilobit</a> (kb) is 1000 <a href="/wiki/Bit" title="Bit">bits</a></li>
<li>one <a href="/wiki/Kilobyte" title="Kilobyte">kilobyte</a> (kB) is 1000 <a href="/wiki/Byte" title="Byte">bytes</a></li>
<li>one kiloohm is (kΩ) is 1000 <a href="/wiki/Ohm" title="Ohm">ohms</a></li></ul>
<h2><span class="mw-headline" id="kilobyte">kilobyte</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kilo-&amp;action=edit&amp;section=2" title="Edit section: kilobyte">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>For the kilobyte, a second definition has been in common use in some fields of <a href="/wiki/Computer_science" title="Computer science">computer science</a> and information technology. It uses <i>kilobyte</i> to mean 2<sup>10</sup> bytes (= 1024 bytes), because of the <a href="/wiki/Mathematical_coincidence#Concerning_base_2" title="Mathematical coincidence">mathematical coincidence</a> that 2<sup>10</sup> is approximately 10<sup>3</sup>. The reason for this application is that digital hardware and architectures natively use <a class="mw-redirect" href="/wiki/Base_2" title="Base 2">base 2</a> exponentiation, and not <a class="mw-redirect" href="/wiki/Base_10" title="Base 10">decimal</a> systems. <a href="/wiki/JEDEC_memory_standards" title="JEDEC memory standards">JEDEC memory standards</a> still permit this definition, but acknowledge the correct SI usage.
</p><p>NIST comments on the confusion caused by these contrasting definitions: "Faced with this reality, the <i>IEEE Standards Board</i> decided that IEEE standards will use the conventional, internationally adopted, definitions of the SI prefixes", instead of kilo for 1024.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>  To address this conflict, a new set of <a href="/wiki/Binary_prefix" title="Binary prefix">binary prefixes</a> has been introduced, which is based on powers of 2. Therefore, 1024 bytes are defined as one <a href="/wiki/Kibibyte" title="Kibibyte">kibibyte</a> (1 KiB).
</p>
<h2><span class="mw-headline" id="Exponentiation">Exponentiation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kilo-&amp;action=edit&amp;section=3" title="Edit section: Exponentiation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>When units occur in <a href="/wiki/Exponentiation" title="Exponentiation">exponentiation</a>, such as in square and cubic forms, any multiplier prefix is considered part of the unit, and thus included in the exponentiation.
</p>
<ul><li>1 km<sup>2</sup> means one square kilometre or the area of a <a class="mw-redirect" href="/wiki/Square_(geometry)" title="Square (geometry)">square</a> that measures 1000 m on each side or 10<sup>6</sup> m<sup>2</sup> (as opposed to 1000 square meters, which is the area of a square that measures 31.6 m on each side).</li>
<li>1 km<sup>3</sup> means one cubic kilometre or the volume of a <a href="/wiki/Cube" title="Cube">cube</a> that measures 1000 m on each side or 10<sup>9</sup> m<sup>3</sup> (as opposed to 1000 cubic meters, which is the volume of a cube that measures 10 m on each side).</li></ul>
