<p>In chemistry, an <b>amphoteric</b> compound is a molecule or ion that can react both as an <a href="/wiki/Acid" title="Acid">acid</a> as well as a <a href="/wiki/Base_(chemistry)" title="Base (chemistry)">base</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>  Many metals (such as <a href="/wiki/Copper" title="Copper">copper</a>, <a href="/wiki/Zinc" title="Zinc">zinc</a>, <a href="/wiki/Tin" title="Tin">tin</a>, <a href="/wiki/Lead" title="Lead">lead</a>, <a href="/wiki/Aluminium" title="Aluminium">aluminium</a>, and <a href="/wiki/Beryllium" title="Beryllium">beryllium</a>) form amphoteric oxides or hydroxides. Amphoterism depends on the <a class="mw-redirect" href="/wiki/Oxidation_states" title="Oxidation states">oxidation states</a> of the oxide. Al<sub>2</sub>O<sub>3</sub> is an example of an amphoteric oxide.
</p><p>The prefix of the word 'amphoteric' is derived from a Greek prefix amphi-, which means both. In chemistry, an amphoteric substance is a substance that has the ability to act either as an acid or a base. Remember that acids donate protons (or accept electron pairs) and bases accept protons. Amphoteric substances can do either. So, we can think of an amphoteric substance as something like a double agent. Just like how a double agent can act as an ally or an enemy depending on the situation, the amphoteric substance can act either as an acid or a base, depending on what other substances it is reacting with.
</p><p><a class="mw-redirect" href="/wiki/Metal_oxide" title="Metal oxide">Metal oxides</a> which react with both acids as well as bases to produce salts and water are known as amphoteric oxides.  Amphoteric oxides include <a href="/wiki/Lead(II)_oxide" title="Lead(II) oxide">lead oxide</a> and <a href="/wiki/Zinc_oxide" title="Zinc oxide">zinc oxide</a>, among many others.
</p><p>One type of amphoteric species are <b>amphiprotic</b> molecules, which can either donate or accept a <a href="/wiki/Proton" title="Proton">proton</a> (H<sup>+</sup>). Examples include <a href="/wiki/Amino_acid" title="Amino acid">amino acids</a> and <a href="/wiki/Protein" title="Protein">proteins</a>, which have <a href="/wiki/Amine" title="Amine">amine</a> and <a href="/wiki/Carboxylic_acid" title="Carboxylic acid">carboxylic acid</a> groups, and <a href="/wiki/Molecular_autoionization" title="Molecular autoionization">self-ionizable compounds</a> such as <a href="/wiki/Water" title="Water">water</a>.
</p><p><b>Ampholytes</b> are amphoteric molecules that contain both acidic and basic groups and will exist mostly as <a href="/wiki/Zwitterion" title="Zwitterion">zwitterions</a> in a certain range of <a href="/wiki/PH" title="PH">pH</a>. The pH at which the average charge is zero is known as the molecule's <i><a href="/wiki/Isoelectric_point" title="Isoelectric point">isoelectric point</a></i>.
Ampholytes are used to establish a stable pH gradient for use in <a href="/wiki/Isoelectric_focusing" title="Isoelectric focusing">isoelectric focusing</a>.
</p>

<h2><span class="mw-headline" id="Etymology">Etymology</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Amphoterism&amp;action=edit&amp;section=1" title="Edit section: Etymology">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>Amphoteric</b> is derived from the Greek word <i lang="el-Latn" title="Greek language text">amphoteroi</i> (<span lang="el" title="Greek language text">ἀμφότεροι</span>) meaning "both".  Related words in acid-base chemistry are <b>amphichromatic</b> and <b>amphichroic</b>, both describing substances such as <a href="/wiki/PH_indicator" title="PH indicator">acid-base indicators</a> which give one colour on reaction with an acid and another colour on reaction with a base.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>
</p>
<h2><span class="mw-headline" id="Amphiprotic_molecules">Amphiprotic molecules</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Amphoterism&amp;action=edit&amp;section=2" title="Edit section: Amphiprotic molecules">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>According to the <a class="mw-redirect" href="/wiki/Br%C3%B8nsted-Lowry_theory" title="Brønsted-Lowry theory">Brønsted-Lowry theory</a> of acids and bases: acids are proton donors and bases are proton acceptors.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> An amphiprotic molecule (or ion) can either donate or accept a <a href="/wiki/Proton" title="Proton">proton</a>, thus acting either as an <a href="/wiki/Acid" title="Acid">acid</a> or a <a href="/wiki/Base_(chemistry)" title="Base (chemistry)">base</a>. <a href="/wiki/Water" title="Water">Water</a>, <a href="/wiki/Amino_acid" title="Amino acid">amino acids</a>, <a href="/wiki/Bicarbonate" title="Bicarbonate">hydrogen carbonate</a> ions and <a class="mw-redirect" href="/wiki/Hydrogen_sulfate" title="Hydrogen sulfate">hydrogen sulfate</a> ions are common examples of amphiprotic species. Since they can donate a proton, all amphiprotic substances contain a hydrogen atom. Also, since they can act like an acid or a base, they are amphoteric.
</p>
<h3><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Amphoterism&amp;action=edit&amp;section=3" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A common example of an amphiprotic substance is the hydrogen carbonate ion, which can act as a base:
</p>
<dl><dd>HCO<sub>3</sub><sup>−</sup> + H<sub>3</sub>O<sup>+</sup> → H<sub>2</sub>CO<sub>3</sub> + H<sub>2</sub>O</dd></dl>
<p>or as an acid:
</p>
<dl><dd>HCO<sub>3</sub><sup>−</sup> + OH<sup>−</sup> → CO<sub>3</sub><sup>2−</sup> + H<sub>2</sub>O</dd></dl>
<p>Thus, it can effectively accept or donate a proton.
</p><p>Water is the most common example, acting as a base when reacting with an acid such as <a href="/wiki/Hydrogen_chloride" title="Hydrogen chloride">hydrogen chloride</a>:
</p>
<dl><dd>H<sub>2</sub>O + HCl → H<sub>3</sub>O<sup>+</sup> + Cl<sup>−</sup>,</dd></dl>
<p>and acting as an acid when reacting with a base such as <a href="/wiki/Ammonia" title="Ammonia">ammonia</a>:
</p>
<dl><dd>H<sub>2</sub>O + NH<sub>3</sub> → NH<sub>4</sub><sup>+</sup> + OH<sup>−</sup></dd></dl>
<h3><span class="mw-headline" id="Not_all_amphoteric_substances_are_amphiprotic">Not all amphoteric substances are amphiprotic</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Amphoterism&amp;action=edit&amp;section=4" title="Edit section: Not all amphoteric substances are amphiprotic">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Although an amphiprotic species must be amphoteric, the converse is not true. For example, the metal oxide ZnO contains no hydrogen and cannot donate a proton. Instead it is a <a class="mw-redirect" href="/wiki/Lewis_acid" title="Lewis acid">Lewis acid</a> whose Zn atom accepts an electron pair from the base OH<sup>−</sup>. The other metal oxides and hydroxides mentioned above also function as Lewis acids rather than Brønsted acids.
</p>
<h2><span id="Amphoteric_oxides_and_hydroxides.5B4.5D"></span><span class="mw-headline" id="Amphoteric_oxides_and_hydroxides[4]">Amphoteric oxides and hydroxides<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Amphoterism&amp;action=edit&amp;section=5" title="Edit section: Amphoteric oxides and hydroxides[4]">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Amphoteric_oxides">Amphoteric oxides</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Amphoterism&amp;action=edit&amp;section=6" title="Edit section: Amphoteric oxides">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Zinc_oxide" title="Zinc oxide">Zinc oxide</a> (ZnO) reacts with both acids and with bases:
</p>
<ul><li>In acid: ZnO + H<sub>2</sub>SO<sub>4</sub> → ZnSO<sub>4</sub> + H<sub>2</sub>O</li>
<li>In base: ZnO + 2 NaOH + H<sub>2</sub>O → Na<sub>2</sub>[Zn(OH)<sub>4</sub>]</li></ul>
<p>This reactivity can be used to separate different <a class="mw-redirect" href="/wiki/Cation" title="Cation">cations</a>, such as zinc(II), which dissolves in base, from manganese(II), which does not dissolve in base.
</p><p><a href="/wiki/Lead(II)_oxide" title="Lead(II) oxide">Lead oxide</a> (PbO):
</p>
<ul><li>In acid: PbO + 2 HCl → PbCl<sub>2</sub> + H<sub>2</sub>O</li>
<li>In base: PbO + 2 NaOH + H<sub>2</sub>O → Na<sub>2</sub>[Pb(OH)<sub>4</sub>]</li></ul>
<p><a href="/wiki/Aluminium_oxide" title="Aluminium oxide">Aluminium oxide</a> (Al<sub>2</sub>O<sub>3</sub>)
</p>
<ul><li>In acid: Al<sub>2</sub>O<sub>3</sub> + 6 HCl→ 2 AlCl<sub>3</sub> + 3 H<sub>2</sub>O</li>
<li>In base: Al<sub>2</sub>O<sub>3</sub> + 2 NaOH + 3 H<sub>2</sub>O → 2 Na[Al(OH)<sub>4</sub>] (hydrated <a href="/wiki/Sodium_aluminate" title="Sodium aluminate">sodium aluminate</a>)</li></ul>
<p><a class="mw-redirect" href="/wiki/Stannous_oxide" title="Stannous oxide">Stannous oxide</a> (SnO)
</p>
<ul><li>In acid : SnO +2 HCl ⇌ SnCl<sub>2</sub> + H<sub>2</sub>O</li>
<li>In base : SnO +4 NaOH + H<sub>2</sub>O ⇌ Na<sub>4</sub>[Sn(OH)<sub>6</sub>]</li></ul>
<p>Some other elements which form amphoteric oxides are <a href="/wiki/Gallium" title="Gallium">gallium</a>, <a href="/wiki/Indium" title="Indium">indium</a>, <a href="/wiki/Scandium" title="Scandium">scandium</a>, <a href="/wiki/Titanium" title="Titanium">titanium</a>, <a href="/wiki/Zirconium" title="Zirconium">zirconium</a>, <a href="/wiki/Vanadium" title="Vanadium">vanadium</a>, <a href="/wiki/Chromium" title="Chromium">chromium</a>, <a href="/wiki/Iron" title="Iron">iron</a>, <a href="/wiki/Cobalt" title="Cobalt">cobalt</a>, <a href="/wiki/Copper" title="Copper">copper</a>, <a href="/wiki/Silver" title="Silver">silver</a>, <a href="/wiki/Gold" title="Gold">gold</a>, <a href="/wiki/Germanium" title="Germanium">germanium</a>, <a href="/wiki/Antimony" title="Antimony">antimony</a>, <a href="/wiki/Bismuth" title="Bismuth">bismuth</a>,  and <a href="/wiki/Tellurium" title="Tellurium">tellurium</a>.
</p>
<h3><span class="mw-headline" id="Amphoteric_hydroxides">Amphoteric hydroxides</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Amphoterism&amp;action=edit&amp;section=7" title="Edit section: Amphoteric hydroxides">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Aluminium_hydroxide" title="Aluminium hydroxide">Aluminium hydroxide</a> is also amphoteric:
</p>
<ul><li>As a base (neutralizing an acid): Al(OH)<sub>3</sub>  +  3 HCl  →   AlCl<sub>3</sub>  +  3 H<sub>2</sub>O</li>
<li>As an acid (neutralizing a base): Al(OH)<sub>3</sub> + NaOH → Na[Al(OH)<sub>4</sub>]</li></ul>
<p><a href="/wiki/Beryllium_hydroxide" title="Beryllium hydroxide">Beryllium hydroxide</a>
</p>
<ul><li>with acid: Be(OH)<sub>2</sub> + 2 HCl → BeCl<sub>2</sub> + 2 H<sub>2</sub>O</li>
<li>with base: Be(OH)<sub>2</sub> + 2 NaOH → Na<sub>2</sub>[Be(OH)<sub>4</sub>].<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup></li></ul>
