<h1 class="firstHeading" id="firstHeading" lang="en">Point source pollution</h1> <div class="mw-body-content" id="bodyContent">
<div class="noprint" id="siteSub">From Wikipedia, the free encyclopedia</div> <div id="contentSub"></div>
<div id="jump-to-nav"></div> <a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
<a class="mw-jump-link" href="#p-search">Jump to search</a>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><div class="hatnote navigation-not-searchable" role="note">For other uses, see <a class="mw-disambig" href="/wiki/Point_source_(disambiguation)" title="Point source (disambiguation)">Point source (disambiguation)</a>.</div>
<div class="thumb tright"><div class="thumbinner" style="width:152px;"><a class="image" href="/wiki/File:Common_Point_Source_Discharges_-_EPA_2010.png"><img alt="" class="thumbimage" data-file-height="642" data-file-width="640" height="150" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b7/Common_Point_Source_Discharges_-_EPA_2010.png/150px-Common_Point_Source_Discharges_-_EPA_2010.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b7/Common_Point_Source_Discharges_-_EPA_2010.png/225px-Common_Point_Source_Discharges_-_EPA_2010.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b7/Common_Point_Source_Discharges_-_EPA_2010.png/300px-Common_Point_Source_Discharges_-_EPA_2010.png 2x" width="150"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/File:Common_Point_Source_Discharges_-_EPA_2010.png" title="Enlarge"></a></div>Water pollution point sources</div></div></div>
<div class="thumb tright"><div class="thumbinner" style="width:153px;"><a class="image" href="/wiki/File:AirPollutionSource.jpg"><img alt="" class="thumbimage" data-file-height="203" data-file-width="151" height="203" src="//upload.wikimedia.org/wikipedia/commons/b/b9/AirPollutionSource.jpg" width="151"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/File:AirPollutionSource.jpg" title="Enlarge"></a></div>Air pollution point sources</div></div></div>
<p>A <b>point source of pollution</b> is a single identifiable source of <a href="/wiki/Air_pollution" title="Air pollution">air</a>, <a href="/wiki/Water_pollution" title="Water pollution">water</a>, <a href="/wiki/Thermal_pollution" title="Thermal pollution">thermal</a>, <a href="/wiki/Noise_pollution" title="Noise pollution">noise</a> or <a href="/wiki/Light_pollution" title="Light pollution">light</a> <a href="/wiki/Pollution" title="Pollution">pollution</a>. A <a href="/wiki/Point_source" title="Point source">point source</a> has negligible extent, distinguishing it from other pollution source geometries. The sources are called <i>point sources</i> because in mathematical modeling, they can be approximated as a mathematical <a href="/wiki/Point_(geometry)" title="Point (geometry)">point</a> to simplify analysis.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> Pollution point sources are identical to other <a href="/wiki/Physics" title="Physics">physics</a>, <a href="/wiki/Engineering" title="Engineering">engineering</a>, <a href="/wiki/Optics" title="Optics">optics</a>, and <a href="/wiki/Chemistry" title="Chemistry">chemistry</a> point sources and include: 
</p>
<ul><li><a class="mw-redirect" href="/wiki/Air_pollution_dispersion_terminology#Air_pollutant_emission" title="Air pollution dispersion terminology">Air pollution</a> from an industrial source (rather than an airport or a road, considered a <a href="/wiki/Line_source" title="Line source">line source</a>, or a forest fire, which is considered an <a href="/wiki/Area_source_(pollution)" title="Area source (pollution)">area source</a>, or <a href="/wiki/Volume_source_(pollution)" title="Volume source (pollution)">volume source</a>)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li><a href="/wiki/Water_pollution" title="Water pollution">Water pollution</a> from factories, power plants, municipal <a href="/wiki/Sewage_treatment" title="Sewage treatment">sewage treatment</a> plants and some farms (<i>see</i> <a href="/wiki/Concentrated_animal_feeding_operation" title="Concentrated animal feeding operation">concentrated animal feeding operation</a>).<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> The U.S. <a href="/wiki/Clean_Water_Act" title="Clean Water Act">Clean Water Act</a> also defines municipal separate <a class="mw-redirect" href="/wiki/Storm_sewer" title="Storm sewer">storm sewer</a> systems and industrial <a href="/wiki/Stormwater" title="Stormwater">stormwater</a> discharges (such as <a class="mw-redirect" href="/wiki/Construction_site" title="Construction site">construction sites</a>) as point sources.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup></li>
<li><a href="/wiki/Noise_pollution" title="Noise pollution">Noise pollution</a> from a <a href="/wiki/Jet_engine" title="Jet engine">jet engine</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup></li>
<li>Disruptive <a class="mw-redirect" href="/wiki/Seismic" title="Seismic">seismic</a> vibration from a localized seismic study<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup></li>
<li><a href="/wiki/Light_pollution" title="Light pollution">Light pollution</a> from an intrusive <a href="/wiki/Street_light" title="Street light">street light</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup></li>
<li><a href="/wiki/Radio" title="Radio">Radio</a> emissions from an interference-producing electrical device</li></ul>
