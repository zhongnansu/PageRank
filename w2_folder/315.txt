<p><b>Lime</b>, also called <b>lime green</b>, <b>lime-green</b>, or <b>bitter lime</b>,<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (September 2017)">citation needed</span></a></i>]</sup> is a <a href="/wiki/Color" title="Color">color</a> that is a shade of green, so named because it is a representation of the color of the citrus fruit called <a href="/wiki/Lime_(fruit)" title="Lime (fruit)">limes</a>.  It is the color that is in between the web color <a href="/wiki/Chartreuse_(color)" title="Chartreuse (color)">chartreuse</a> and yellow on the <a href="/wiki/Color_wheel" title="Color wheel">color wheel</a>.<sup class="reference" id="cite_ref-mp_2-1"><a href="#cite_note-mp-2">[2]</a></sup>
</p>

<h2><span id="Lime_.28traditional_lime_green.29"></span><span class="mw-headline" id="Lime_(traditional_lime_green)">Lime (traditional lime green)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Lime_(color)&amp;action=edit&amp;section=1" title="Edit section: Lime (traditional lime green)">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The first recorded use of <i>lime green</i> as a colour name in <a href="/wiki/English_language" title="English language">English</a> was in 1890.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-mp_2-2"><a href="#cite_note-mp-2">[2]</a></sup>
</p><p>Lime (color hex code #BFFF00) is a pure <a href="/wiki/Spectral_color" title="Spectral color">spectral color</a> at approximately 564 nanometers on the <a href="/wiki/Visible_spectrum" title="Visible spectrum">visible spectrum</a> when plotted on the <a class="mw-redirect" href="/wiki/CIE_chromaticity_diagram" title="CIE chromaticity diagram">CIE chromaticity diagram</a>.
</p>

<p><b>Lemon-Lime</b> is a <a href="/wiki/Fluorescence" title="Fluorescence">fluorescent</a> chartreuse color that is named after the carbonated soft drinks such as <a href="/wiki/Sprite_(soft_drink)" title="Sprite (soft drink)">Sprite</a>, <a href="/wiki/7_Up" title="7 Up">7 Up</a> and <a href="/wiki/Sierra_Mist" title="Sierra Mist">Sierra Mist</a>.
</p><p>The red value to this neon color is almost to <a href="/wiki/Yellow" title="Yellow">yellow</a>.
</p>
